## 功能介绍

一个工具类小应用。

提供给轮班上班的人们，上班时间不按照周一至周五工作日，周六日休息，有可能两天白班两天夜班三天晚班然后休息两天，然后再循环。周而复始一如既往，不管周六日，哪管黄金周。

还掺杂了其他小功能，天气预报、笑话大全、影讯，调用了`聚合数据`提供的接口。

还有注册、登录、记录事项等功能，使用了`Bmob后端服务`
